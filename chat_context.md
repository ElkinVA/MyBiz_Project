# Контекст чата для ИИ-ассистента

## Как это работает:
1. Когда вы говорите **"запомни это"**, я создаю краткое резюме текущей беседы
2. Сохраняю его в файл `/workspace/chat_context.md`
3. В начале каждого нового чата вы можете прикрепить этот файл или скопировать его содержимое
4. Я прочитаю контекст и продолжу диалог с учётом предыдущих обсуждений

## Формат команды:
- `"запомни это"` — сохранить текущий контекст
- `"запомни это: [дополнительная инструкция]"` — сохранить с уточнением
- `"покажи контекст"` — вывести текущее сохранённое резюме
- `"очисти контекст"` — удалить сохранённую историю

## Что сохраняется:
- Ваши ключевые инструкции и требования
- Мои действия и решения
- Важные технические детали
- Принятые решения по архитектуре
